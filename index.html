Blender Addons

  • ADH Rigging Tools. Tools for my personal rigging needs, automating most mind-numbing tasks in setting up a rig.
  • Import Rig. A simple tool to automatically import a rig and its script. Without this, some animator won't bother importing a rig's script, which could make the rig harder to use.
  • OHA Animation Tools. Tools for animation workflow, made for OHA Studios.
  • OHA Layout Tools. A tool for automated creation of layout files from animatics file, made for OHA Studios.
  • Display Matrix. A developer tool to show transformation matrices of all selected bones/objects.
